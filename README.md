# Clima_JS_API_Integrada
# 🌤️ Clima

App de previsão do tempo com design moderno e autocomplete inteligente integrado a API de clima.

## 🚀 Funcionalidades
- 🔍 Busca com sugestões automáticas (cidade + país + estado)
- 🌡️ Temperatura, sensação térmica, umidade, vento e precipitação
- ⏰ Relógio digital em tempo real
- 📱 Design responsivo e animações suaves

## 🛠️ Tecnologias
- HTML5, CSS3, JavaScript ES6
- API Open-Meteo (clima e geocoding)

## 📦 Como usar
1. Abra o index.html no navegador
2. Digite o nome de uma cidade
3. Selecione a sugestão desejada
4. Veja as informações climáticas

## 🎨 Destaques
- Interface com glassmorphism
- Efeito de nuvens animadas
- Autocomplete com informações de localização
- Auxilio do DeepSeek somente para ajuda com design mais moderno

## 📱 Responsivo
Funciona perfeitamente em desktop, tablet e mobile!
